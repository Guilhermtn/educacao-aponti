# 🎓 EvoluiTI — Plataforma de Mentoria para TI

> Projeto desenvolvido com foco na **ODS 4 – Educação de Qualidade**

---

## 📌 Problema Identificado

Alto índice de reprovações e trancamentos na disciplina de **Introdução à Computação** no ensino superior, evidenciando a necessidade de suporte pedagógico acessível e especializado para estudantes da área de TI.

---

## 💡 Solução Proposta

Um **portal colaborativo de mentorias** no modelo "Cambly para TI", conectando estudantes de nível superior com instrutores especializados.

- **Aluno** cadastra suas necessidades de mentoria na plataforma.
- **Instrutor** visualiza os pedidos e aceita demandas conforme sua área de expertise.

---

## 🛠️ Stack Tecnológica

| Camada | Tecnologia |
|---|---|
| Frontend | React + Next.js |
| Tempo real | WebRTC + WebSockets |
| Backend | Node.js |
| Banco de dados | PostgreSQL + Redis |
| Infraestrutura | AWS |

---

## ✅ Funcionalidades

- **Mentorias individuais e em grupo** com especialistas de tecnologia
- **Escolha de mentores** por área de atuação e experiência
- **Agendamento flexível (24/7)** via web e aplicativo
- **Trilhas de aprendizagem estruturadas** para carreiras e certificações em TI
- **Gravação das sessões** para revisão posterior
- **Transcrição automática** de aulas, mentorias e discussões técnicas
- **Feedback técnico personalizado** sobre código, arquitetura e boas práticas
- **Desafios e atividades práticas** adaptados aos objetivos do profissional
- **Acompanhamento de progresso** com métricas de evolução e competências
- **Recursos de IA** para revisão de código, geração de resumos e recomendações de aprendizado
